## 📋 系統工作流程

### 定期檢查流程 (每 6 小時自動執行)

```
時間: 每 6 小時
┌────────────────────────────────┐
│ Scheduler 觸發 check_all()     │
└────────────┬───────────────────┘
             │
             ↓
┌────────────────────────────────┐
│ Scraper 連接 RSS feed           │
│ 獲取最新 XML 內容              │
└────────────┬───────────────────┘
             │
             ↓
┌────────────────────────────────┐
│ 解析 XML 提取新聞列表          │
│ 組合成文字快照                 │
└────────────┬───────────────────┘
             │
             ↓
┌────────────────────────────────┐
│ 計算 SHA-256 hash              │
│ 與上次存儲的 hash 比較         │
└────────────┬───────────────────┘
             │
        ┌────┴────┐
        │          │
    相同 ↓          ↓ 不同
        │          │
     無更新    ┌─────────────┐
              │ Diff Service │
              │ 比對差異    │
              └────┬────────┘
                   │
                   ↓
        ┌────────────────────┐
        │ 生成摘要           │
        │ 提取新增/移除內容  │
        └────┬───────────────┘
             │
             ↓
        ┌────────────────────┐
        │ 建立 Notification  │
        │ 發送郵件          │
        └────┬───────────────┘
             │
             ↓
        用戶收到通知
```

### 通知內容示例

