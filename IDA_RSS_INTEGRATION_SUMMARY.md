## 📞 技術支援

### 常見問題解決

**Q: 為什麼沒有收到通知?**
```
A: 檢查以下:
   1. 應用是否在運行 (python app.py)
   2. 檢查頻率設定 (預設 6 小時)
   3. 手動觸發檢查測試
   4. 查看應用日誌
```

**Q: 如何修改檢查頻率?**
```
A: 在訂閱設定中改 check_interval_minutes:
   60 = 每小時
   1440 = 每天
   10080 = 每週
```

**Q: 可以監控特定內容嗎?**
```
A: 可以，設定 watch_description 為:
   "只監控 AI 和知識經濟產業相關新聞"
   系統會用 AI 提取相關內容進行比對
```
